# IRabi — Платформа экспертов и пользователей

IRabi — веб-платформа, соединяющая экспертов и пользователей. Эксперты публикуют свои слоты времени для консультаций, а пользователи могут забронировать доступные слоты.

## Роли пользователей

| Роль | Описание |
|------|----------|
| **Администратор** | Системные настройки, техническое управление |
| **Владелец** | Полное управление платформой и пользователями |
| **Эксперт** | Публикация слотов времени, управление расписанием |
| **Пользователь** | Бронирование слотов, оплата |

## Ключевые сущности

- **Слот времени** — доступное время эксперта для консультации
  - Дата и время, длительность, стоимость
  - Онлайн/офлайн, место проведения
  - Статус: свободен → забронирован → проведён / отменён
- **Бронирование** — запись пользователя на слот эксперта
  - Связывает пользователя со слотом
  - Статус: pending → confirmed → completed / cancelled

## Технологии

- **Backend:** PHP 8.x + Garnet Framework
- **Frontend:** TypeScript + Rspack
- **Database:** MySQL
- **Templates:** Twig
- **CSS:** Less + Bootstrap 5
