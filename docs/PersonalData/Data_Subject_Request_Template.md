# Шаблон ответа на запрос субъекта персональных данных

## 1. Регистрация запроса

Все запросы субъектов ПДн фиксируются в Журнале учёта обращений (Приложение 4 Положения):

- дата и способ поступления;
- ФИО заявителя, контактные данные;
- содержание запроса;
- срок ответа;
- результат и дата ответа.

## 2. Сроки

- Подтверждение факта обработки ПДн / предоставление сведений: **не более 10 рабочих дней** с возможностью продления до **5 рабочих дней** с уведомлением заявителя.
- Уточнение, блокирование, уничтожение ПДн при подтверждении неточности или незаконности обработки: **в течение 7 рабочих дней** со дня предоставления подтверждающих документов.
- При отзыве согласия — прекращение обработки и уничтожение ПДн: **в течение 30 дней** при отсутствии иных правовых оснований.

## 3. Идентификация заявителя

Перед предоставлением сведений необходимо идентифицировать заявителя. Допустимые способы:

- запрос направлен с e-mail, привязанного к учётной записи;
- запрос содержит копию документа, удостоверяющего личность;
- запрос подписан УКЭП.

При невозможности идентифицировать заявителя — запрашивать дополнительные сведения, не раскрывая ПДн до подтверждения личности.

## 4. Технический регламент исполнения

Для запросов на **выгрузку всех ПДн пользователя** реализовать в админ-панели:

- кнопка «Экспорт ПДн» в карточке пользователя — формирует ZIP с JSON всех записей по `account_id` (профиль, бронирования, сообщения, платежи, логи);
- кнопка «Удалить аккаунт и ПДн» — каскадное удаление либо обезличивание (замена имени на `deleted_user_<id>`, очистка e-mail, обнуление IP в логах).

Для пользователя на сайте — отдельная страница `/account/privacy` со ссылками «Скачать мои данные» и «Удалить аккаунт».
