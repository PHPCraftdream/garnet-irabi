# Архитектура приложения

## Структура бандлов

Приложение разделено на логические бандлы:

### Common
Общий код, используемый всеми бандлами:
- Определения таблиц БД (`Tables/`)
- Конфигурации сущностей (`Params/`)
- Переводы (`I18n/`)
- Общие Twig-шаблоны

### Foreground
Основной интерфейс приложения:
- Главная страница
- Регистрация и авторизация
- Просмотр экспертов и слотов
- Бронирование слотов
- Личный кабинет эксперта (`TeachingController`)
- Профиль эксперта (`ExpertController`)

### Dashboard
Административная панель:
- Управление пользователями
- Модерация экспертов (одобрение/отзыв)
- Бронирования и финансовые отчёты
- Логи действий и системные настройки

## Контроллеры

### MainController
- URL: `/`
- Главная страница приложения
- Доступен всем авторизованным пользователям

### RegisterController
- URL: `/register`
- Регистрация новых пользователей
- Не требует авторизации

### TeachingController
- URL: `/expert`
- Управление слотами и расписанием
- Доступен только экспертам (middleware `expertOnly`)

### ExpertController
- URL: `/expert`
- Публичный профиль эксперта
- Доступен всем авторизованным пользователям

### SlotsController
- URL: `/slots`
- Просмотр доступных слотов
- Бронирование слотов пользователями

### BookingsController
- URL: `/bookings`
- Управление бронированиями
- Доступен всем авторизованным пользователям

### AccountsController
- URL: `/accounts`
- Управление аккаунтами пользователей
- Доступен только модераторам

## Frontend структура

```
FrontBuilder/IRabi/
├── Common/           # Общие стили и скрипты
│   ├── Styles/       # Less-файлы
│   └── Scripts/      # TypeScript, I18n
├── Dashboard/        # Скрипты админ-панели
│   ├── Styles/
│   └── Scripts/
└── Foreground/       # Скрипты основного интерфейса
    ├── Styles/
    ├── Scripts/
    │   ├── Foreground.ts
    │   └── Registration/
    └── Assets/
```

## Миграции

Миграции находятся в `Migrations/Items/`:

- `M_0002.php` — создание таблиц
- `M_0003.php` — изменения структуры
- `M_0004.php` — добавление полей

Запуск миграций:
```bash
cd Apps/IRabi
php run_cmd.php migrate
```

## Конфигурация

Конфигурационные файлы в `WorkDir/`:

| Файл | Описание |
|------|----------|
| `app.ini` | Основные настройки (baseUrl, title) |
| `db.ini` | Подключение к БД |
| `email.ini` | Настройки почты |

## Сборка frontend

```bash
cd FrontBuilder
npm run build
```

Генерация I18n файлов:
```bash
php PrepareParams.php
```
