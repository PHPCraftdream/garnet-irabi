# Бизнес-аналитика IRabi

## 1. Общее описание системы

IRabi — платформа для организации консультаций между экспертами и пользователями. Система позволяет экспертам публиковать слоты свободного времени, а пользователям — бронировать и оплачивать консультации.

## 2. Целевая аудитория

| Роль | Цель использования |
|------|-------------------|
| **Администратор** | Управление системными настройками, техническая поддержка |
| **Владелец** | Полное управление платформой, всеми пользователями и контентом |
| **Эксперт** | Публикация слотов, ведение экспертной деятельности |
| **Пользователь** | Поиск экспертов, бронирование слотов, оплата консультаций |

## 3. Ключевые бизнес-сущности

### 3.1 Слот времени
Доступное время эксперта для консультации.

**Атрибуты:**
- Дата и время начала
- Длительность (в минутах)
- Стоимость
- Формат (онлайн/офлайн)
- Место проведения
- Вместимость (макс. пользователей)
- Статус (свободен/забронирован/проведён/отменён)

### 3.2 Бронирование
Запись пользователя на слот эксперта.

**Атрибуты:**
- Связь с пользователем и слотом
- Статус (pending/confirmed/completed/cancelled)
- Дата создания

## 5. Бизнес-процессы

### 5.1 Жизненный цикл слота времени

```
[Создание слота] → [Свободен] → [Забронирован] → [Проведён]
                                    ↓
                              [Отменён]
```

### 5.2 Процесс бронирования пользователем

```
[Поиск эксперта/слота] → [Выбор слота] → [Оплата] → [Подтверждение] → [Консультация]
```
